# Projects

Trajectory prediction based on AutoML and FL

2020.10 - present

By analyzing and summarizing the existing trajectory prediction (TP) model, using automated machine learning (AutoML) technology, the optimal TP model is automatically designed under the framework of federated learning (FL), so that the algorithm can effectively use distributed and private data sets to Design a more general TP model. Paper based on this project have been submitted.

Chunnan W., Xiang C., et al

[PDF] [arXiv]

Heilongjiang Provincial Center for Disease Control and Prevention, Foodborne Disease Project

2019.12 - 2020.06

This subject belongs to the ”Mobile Terminal for On-site Epidemiological Investigation of Food Safety Accidents” project of the Heilongjiang Provincial Center for Disease Control and Prevention. It provides a theoretical and practical basis for the completion of data collection functions, prediction and discrimination functions, and laboratory test data entry functions.

Real-time monitoring and self-healing system for thermal power system failures

2018.11 - 2019.09

Propose fault-oriented data preparation methods, intelligent detection methods based on semi-supervised learning, and use multi-dimensional time series analysis and pattern recognition and automatic recovery based on knowledge automation to realize intelligent detection and automatic recovery systems for power grid failures and establish demonstration applications to achieve traditional Approach to artificial intelligence-driven transformation.
