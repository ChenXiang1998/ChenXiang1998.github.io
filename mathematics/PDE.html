# 偏微分方程

# 一、基本概念

# 1.1 偏微分方程(PDE, partial differential equations)

关于函数u(x,y,⋯)u(x,y,\cdots )u(x,y,⋯)的PDE是形如

F(x,y,⋯,u,ux,uy,⋯,uxx,uxy,⋯)=0F(x,y,\cdots,u,u_x,u_y,\cdots,u_{xx},u_{xy},\cdots) = 0 F(x,y,⋯,u,u​x​​,u​y​​,⋯,u​xx​​,u​xy​​,⋯)=0

的关系式，其中FFF是自变量x,y,⋯x,y,\cdotsx,y,⋯，未知函数uuu以及uuu的有限多个偏导数的已知函数。

# 1.2 解(solutions)

偏微分方程通常在含自变量的定义域内讨论。若在考虑的定义域内存在一多元函数u=φ(x,y,⋯)u=\varphi (x,y,\cdots )u=φ(x,y,⋯)满足偏微分方程，则称uuu为该方程的一个解。特别地有形式解、通解、特解。

# 1.3 阶(order)

方程实际所含未知函数偏导数的最高阶数称为方程的阶。注意要和“次数”做区别。例如：

ut+uux=(ux)3u_{t}+u u_{x}=(u_{x})^3 u​t​​+uu​x​​=(u​x​​)​3​​

是一阶方程而不是三阶方程。

# 二、偏微分方程的基本分类

# 2.1 线性(linear)

如果偏微分方程关于未知函数uuu及其所有的偏导数是线性的，并且其系数仅依赖于自变量x,y,⋯x,y,\cdotsx,y,⋯。也就是说，偏微分方程中未知函数与其各偏导函数仅为一次，并且不含它们的乘积项。如：

ut−uxx+texu=0u_t-u_{xx}+te^xu=0 u​t​​−u​xx​​+te​x​​u=0

在线性方程中，不含未知函数及其偏导数的项称为自由项。若自由项不为零，称方程为非齐次的；若自由项为零，则称方程为齐次的。

# 2.2 拟线性(quasilinear)

如果mmm阶偏微分方程关于未知函数uuu的mmm阶偏导数是线性的，并且其mmm阶偏导数的系数仅依赖于x,y,⋯x,y,\cdotsx,y,⋯以及未知函数uuu的阶数低于mmm的偏导数。拟线性PDE是非线性的，但关于其最高阶偏导数是线性的。如：

uuxy+ux2uy2u2uyy+ux2+uy2+u2=0uu_{xy}+u_x^2u_y^2u^2u_{yy}+u_x^2+u_y^2+u^2=0 uu​xy​​+u​x​2​​u​y​2​​u​2​​u​yy​​+u​x​2​​+u​y​2​​+u​2​​=0

在拟线性方程中，由最高阶偏导数所组成的部分称为方程的主部。上面方程的主部为uuxy+ux2uy2u2uyyuu_{xy}+u_x^2u_y^2u^2u_{yy}uu​xy​​+u​x​2​​u​y​2​​u​2​​u​yy​​。如果方程的主部的各项系数不含未知函数，就称它为半线性方程。如：

xuxx+xyuxy+yuyy+uux+uuy=x2+y2xu_{xx}+xyu_{xy}+yu_{yy}+uu_x+uu_y=x^2+y^2 xu​xx​​+xyu​xy​​+yu​yy​​+uu​x​​+uu​y​​=x​2​​+y​2​​
