# 实习与项目经历

# Spot VM 驱逐预测

微软亚洲研究院(MSRA)，Software Analytics组实习生

2021.12 -- 2022.04

参与AutoML时间序列模型搜索软件包“ts- lib”的代码开发和测试，该软件包可帮助用户选择最佳拟合模型和相应参数，处理缺失数据并识别多个时间序列。

# 基于AutoML与FL的轨迹预测

2020.10 — 2021.11 哈工大海量数据研究中心

通过分析和总结现有轨迹预测(TP)模型，利用自动化机器学习（AutoML）技术，在联邦学习(FL)框架下自动设计最佳TP模型，使算法能够有效地利用分布式和私有数据集来设计更通用的TP模型。论文已被CVPR2022接收。

# 食源性疾病项目

2019.12 — 2020.06 黑龙江省省疾控中心

本课题隶属于黑龙江省疾病预防控制中心“食品安全事故现场流行病学调查 移动终端”项目。负责建立食源性疾病致病因子预测模型，为早期和有效的患者识别和治疗铺平了道路。基于该项目撰写的论文于2021年5月被ICPCSEE录用。

# 基于无监督学习方法的收缩城市识别研究

2018.11 — 2019.04 哈工大经济与管理学院

从粤港澳大湾区城市群中9座地级市的四个维度建立潜在反映收缩城市特征的指标组成的识别体系，利用无监督学习算法划分城市类别,并通过定性分析初步探究了其中收缩城市的形成原因。于2020年5月在北大核心期刊《公共行政评论》发表论文。
