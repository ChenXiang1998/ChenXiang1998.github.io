# 课题与项目

基于 AutoML 与 FL 的轨迹预测

2020.10 - 至今

通过分析和总结现有轨迹预测 (TP) 模型，利用自动化机器学习（AutoML）技术，在联邦学习 (FL) 框 架下自动设计最佳 TP 模型，使算法能够有效地利用分布式和私有数据集来设计更通用的 TP 模型。基 于该项目撰写的论文已投出。

黑龙江省省疾控中心：食源性疾病项目

2019.12 - 2020.06

本课题隶属于黑龙江省疾病预防控制中心“食品安全事故现场流行病学调查移动终端”项目，为完成 数据信息采集功能、预测判别功能、实验室检验数据录入功能等提供理论与实践基础。

火电系统故障实时监测与自愈系统

2018.11 - 2019.09

提出面向故障的数据准备方法、基于半监督学习的智能检测方法，利用多维时间序列分析与模式识别 和基于知识自动化的自动恢复，实现电网故障智能检测和自动恢复系统并建立示范应用，实现由传统 方法向人工智能驱动的转变。
